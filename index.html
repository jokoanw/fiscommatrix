Pembuktian Vektor \(\vec{P}\)

1. Proyeksi Alas (Bidang XY)

Tentukan komponen \(x\) dan \(y\) dari jari-jari silinder \(\rho\) dan sudut \(\phi\):

\[ x = \rho \cos \phi, \quad y = \rho \sin \phi \]

2. Ketinggian (Sumbu Z)

Tambahkan komponen \(z\) untuk membentuk titik \(\vec{P}\) di ruang 3D:

\[ \vec{P} = (\rho \cos \phi)\hat{i} + (\rho \sin \phi)\hat{j} + z\hat{k} \]

3. Vektor Radial \(\hat{\rho}\)

Faktorkan \(\rho\) untuk melihat unit vektor radial pada bidang alas:

\[ \vec{P} = \rho (\hat{i} \cos \phi + \hat{j} \sin \phi) + z\hat{k} \]

4. Kesimpulan Akhir

Ganti komponen arah radial dengan unit vektor \(\hat{\rho}\):

\[ \vec{P} = \rho \hat{\rho} + z\hat{k} \]
Klik tombol Tahap di kiri, dan putar grafik 3D menggunakan mouse/trackpad.
