Pembuktian Vektor Posisi $\vec{P}$

Halo, saya Afdhi (Mainn) dari Fisika UNSOED. Berikut adalah breakdown rumusnya:

1. Transformasi Kartesius $x = \rho \cos \phi$
$y = \rho \sin \phi$
$z = z$
2. Substitusi ke $\vec{P}$ $\vec{P} = ( \rho \cos \phi )\hat{i} + ( \rho \sin \phi )\hat{j} + z\hat{k}$
3. Faktorisasi $\rho$ $\vec{P} = \rho ( \hat{i} \cos \phi + \hat{j} \sin \phi ) + z\hat{k}$
4. Definisi Unit Vektor $\hat{\rho}$

Karena $\hat{\rho} = \cos \phi \hat{i} + \sin \phi \hat{j}$, maka:

$\vec{P} = \rho \hat{\rho} + z\hat{k}$
