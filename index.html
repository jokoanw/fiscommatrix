Pembuktian Vektor Posisi \(\vec{P}\)

Oleh: Afdhi Abdilah (K1C025036) - Fisika UNSOED

1. Proyeksi Kartesius ke Silinder

Berdasarkan geometri pada gambar, komponen \(x\) dan \(y\) dapat dinyatakan dalam radius radial \(\rho\) dan sudut \(\phi\):

\[ x = \rho \cos \phi, \quad y = \rho \sin \phi, \quad z = z \]

2. Substitusi Vektor Posisi Standar

Masukkan komponen tersebut ke dalam vektor posisi Kartesius \(\vec{P} = x\hat{i} + y\hat{j} + z\hat{k}\):

\[ \vec{P} = (\rho \cos \phi)\hat{i} + (\rho \sin \phi)\hat{j} + z\hat{k} \]

3. Definisi Unit Vektor \(\hat{\rho}\)

Dari hubungan unit vektor, kita tahu bahwa \(\hat{\rho} = \hat{i} \cos \phi + \hat{j} \sin \phi\). Mari kita faktorkan \(\rho\):

\[ \vec{P} = \rho \underbrace{(\hat{i} \cos \phi + \hat{j} \sin \phi)}_{\hat{\rho}} + z\hat{k} \]

4. Kesimpulan Akhir

Dengan mengganti bagian dalam kurung dengan unit vektor \(\hat{\rho}\), maka diperoleh:

\[ \vec{P} = \rho \hat{\rho} + z\hat{k} \]
