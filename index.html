Pembuktian Vektor Posisi \(\vec{P}\)

Afdhi Abdilah (K1C025036) - Fisika UNSOED

1. Proyeksi ke Bidang Alas

Komponen \(x\) dan \(y\) dinyatakan dalam radius \(\rho\) dan sudut \(\phi\):

\[ x = \rho \cos \phi, \quad y = \rho \sin \phi, \quad z = z \]

2. Substitusi Vektor Posisi

Masukkan ke vektor \(\vec{P} = x\hat{i} + y\hat{j} + z\hat{k}\):

\[ \vec{P} = (\rho \cos \phi)\hat{i} + (\rho \sin \phi)\hat{j} + z\hat{k} \]

3. Faktorisasi \(\rho\)

\[ \vec{P} = \rho (\hat{i} \cos \phi + \hat{j} \sin \phi) + z\hat{k} \]

4. Substitusi Unit Vektor \(\hat{\rho}\)

Karena \(\hat{\rho} = \hat{i} \cos \phi + \hat{j} \sin \phi\), maka persamaan akhirnya:

\[ \vec{P} = \rho \hat{\rho} + z\hat{k} \]
